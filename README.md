## 课程设计
### 基本内容（成绩范围：“中”、“及格”或“不及格”）
* （1）扩充赋值运算：

    **+= 和 -=**
* （2）扩充语句（Pascal的FOR语句）:

    **①FOR <变量>:=<表达式> TO <表达式> DO <语句>**

    **②FOR <变量>:=<表达式> DOWNTO <表达式> DO <语句>**
     
     其中，语句①的循环变量的步长为1，语句②的循环变量的步长为-1。

### 选做内容（成绩评定范围扩大到：“优”和“良”）
* （**1）增加运算：++ 和 --。**
* （2）增加类型：① 字符类型；          ② 实数类型。
* （3）扩充函数：① 有返回值和返回语句；② 有参数函数。
* （4）增加一维数组类型（可增加指令）。
* **（5） 增加 注释; 注释由/*和*/包含，或由//注释一行；**
* （6）其他典型语言设施。
